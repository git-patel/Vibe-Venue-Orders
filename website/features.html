Comprehensive App Features

Vibe Venue is a highly customized, full-stack Flutter POS application running entirely on your Android device. It acts as both the database, the manager, and the HTTP server all in one.

← Back to Home

Core Architecture

Local Shelf HTTP Server

The app spins up a lightweight Dart `shelf` server operating on port 8080. It dynamically serves an optimized, minified HTML frontend directly to connected attendees via the Android Wi-Fi hotspot.

SQLite Data Storage

All menus, orders, customer details, and settings are stored locally on the device using a high-performance SQLite database. Your data is yours, safe and secure without any cloud dependency.

Zero-Network Connectivity

By connecting to the device's self-generated hotspot, attendees bypass congested 4G/5G networks. They scan a printed QR code and order locally in milliseconds.

Vendor Management Tools

TTS Voice Notifications

Never miss an order. The app uses the device's Text-to-Speech engine to announce new orders aloud (e.g., "New order received. Order ID: ORD-014 from John"). Fully adjustable voice speed and pitch in settings.

Rush Mode & Automation

Is the halftime rush hitting hard? Enable "Rush Mode" to automatically auto-advance orders to "Ready" after a configured number of minutes. You can also configure the app to auto-start preparation instantly.

CSV Data Export

End of shift? The app automatically cleans up delivered orders and can export all historical daily payment data directly to a CSV file. Share it via WhatsApp or Email when you re-connect to the internet.

Built-in Analytics

A dedicated analytics dashboard shows a daily sales chart, top-selling items pie chart, and a breakdown of payments (Cash vs. UPI/Online) so you can track exact revenue.

The Attendee Experience

Advanced Menu Customization

Attendees see a beautiful, responsive HTML menu. They can customize items using rich metadata configured by the vendor: Checkboxes for extras, Dropdowns for sizes, and Text inputs for special requests.

Seat & Zone Ordering

Attendees enter their exact location (e.g., "Block B, Seat 142") and can add a tip during checkout. They see their order ID immediately and can track the status in real-time as the vendor updates it.
